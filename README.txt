Входные параметры подгружаются из файлов *_search.json и *_reverse.json 
	для проведения прямого и обратного геокодирования соответственно.

Для работы с nominatim использовались запросы следующего вида:

https://nominatim.openstreetmap.org/search - прямое
https://nominatim.openstreetmap.org/reverse - обратное

Полученные ответы приводились к формату json (dict), чтобы было удобно вычленять по ключам нужные значения.

Методы, соответствующие прямому и обратному геокодированию:

def test_get_coordinate(self)
def test_get_addres(self)

Используемые библиотеки:

import unittest #данный модуль поддерживает автоматизацию тестов
import requests #для работы с запросами
import json #для работы с json файлами

Из выше приведённенных файлов .json были считаны ожидаемые результаты.
Из ответов на запросы - полученные.
С помощью self.assertEqual() было проведено из сравнение.
